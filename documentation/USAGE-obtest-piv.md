optest-getcert

this retrieves the certificate from a credential that uses ISO 7816-4/NIST SP-800-73-4 messaging.
it takes one argument, the reader number.  reader numbers start at 0.

obtest-challenge

    obtest-challenge <Reader number> <payload-size> [<payload-filename>]

if no payload file is specificed it is filled in with a constant (currently 0x17).
